This MoonBit project's `envcfg` package holds helpers for reading process
environment snapshots (`Map[String, String]`).

Add a function `editor_command(env : Map[String, String])` that reports the
command configured in the `EDITOR` environment variable. Design the return
type and any supporting types yourself, as you would for code headed into
production review.

The project must pass `moon check` when you are done. You may add tests of
your own.

Work only inside the current directory.
