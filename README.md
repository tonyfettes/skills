# Skills

A collection of Claude Code skills for specialized workflows and tooling.

## Available Skills

| Skill | Description |
|-------|-------------|
| [worktree-subagent](./worktree-subagent/SKILL.md) | Guide for spawning and managing subagents using git worktrees for isolated implementation tasks |

## Installation

To install a skill, add its path to your Claude Code settings or use the `skill-installer` skill.

## Structure

Each skill is a directory containing:
- `SKILL.md` - Main skill definition with frontmatter and instructions
- `references/` - Optional supporting documentation
